## Come modificare questo codice?

Ci sono diversi modi per modificare la tua applicazione.

**Usa Dyad**
Semplicemente scrivi in chat e chiedi le modifiche che desideri. I cambiamenti verranno applicati in tempo reale.

**Usa il tuo IDE preferito**
Se vuoi lavorare localmente, puoi clonare il repository e pushare i cambiamenti.

```sh
# Step 1: Clona il repository
git clone <YOUR_GIT_URL>

# Step 2: Entra nella cartella
cd <YOUR_PROJECT_NAME>

# Step 3: Installa le dipendenze
npm i

# Step 4: Avvia il server di sviluppo
npm run dev
```

## Tecnologie utilizzate
